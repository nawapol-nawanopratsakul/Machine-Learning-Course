# Data Preprocessing

This program is used to prepare data before applying Machine Learning. It reads data from a CSV file and checks basic information such as the dataset size, data types, missing values, and duplicate records.

Next, it displays the data using a Histogram and a Correlation Heatmap to help analyze the dataset. After that, it performs data cleaning by removing duplicate records, filling missing values, correcting incorrect data, and converting data types.

Finally, the program applies Label Encoding and One-Hot Encoding to convert the data into a format suitable for Machine Learning. The cleaned dataset is then saved as a new CSV file.
